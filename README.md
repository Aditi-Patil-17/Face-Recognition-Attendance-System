# Face-Recognition-Attendance-System
## Overview

The Face Recognition Attendance System is an AI-powered attendance management solution designed to automate attendance recording using real-time facial recognition. The system utilizes computer vision techniques to identify registered individuals and mark attendance automatically, eliminating the need for manual attendance processes.

The project was developed using Python, OpenCV, and Raspberry Pi, focusing on edge AI deployment and real-time image processing.

---

## Objectives

* Automate attendance recording using facial recognition.
* Reduce manual attendance errors and proxy attendance.
* Develop a low-cost edge AI solution using Raspberry Pi.
* Improve recognition speed and system efficiency.

## System Architecture

1. Camera captures live video frames.
2. Frames are processed using OpenCV.
3. Faces are detected in real-time.
4. Detected faces are matched against the trained database.
5. Attendance is automatically recorded for recognized individuals.
6. Attendance records are stored for future reference.

---

## Implementation

The system was developed using a Raspberry Pi-based architecture where video frames are continuously processed using computer vision algorithms.

The implementation included:

* Image acquisition
* Face detection
* Face recognition
* Database matching
* Attendance logging
* Performance optimization

The processing pipeline was optimized to improve inference speed and reduce recognition latency.

---

## Results

* Successfully automated attendance recording.
* Achieved approximately 58% reduction in recognition latency.
* Improved processing speed from 2.5 seconds to 1.05 seconds.
* Demonstrated reliable real-time face recognition performance.
* Eliminated manual attendance tracking requirements.
